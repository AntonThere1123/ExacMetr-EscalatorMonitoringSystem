# 🚇 Приложение для отслеживания состояния эскалаторов

## О проекте
Приложение автоматизирует процесс приёма жалоб на работу эскалаторов, контроля их состояния и распределения заданий между мастерами.

## Стек технологий
- **Язык:** C# (.NET Windows Forms)
- **Среда:** Visual Studio 2022
- **База данных:** MySQL
- **Библиотека:** MySql.Data

## Роли пользователей

### 👤 Пользователь
- Регистрация и авторизация
- Просмотр эскалаторов и их статуса
- Отправка жалобы на эскалатор
- Просмотр истории жалоб и ответов менеджера

### 👨‍🔧 Мастер
- Просмотр закреплённых эскалаторов
- Просмотр заданий от менеджера
- Изменение статуса задания
- Отправка жалобы

### 👨‍💼 Менеджер
- Просмотр и фильтрация жалоб
- Ответ на жалобы
- Управление статусами эскалаторов
- Регистрация мастеров
- Назначение мастеров на эскалаторы

## Установка и запуск

### 1. Установите MAMP или XAMPP
### 2. Импортируйте базу данных
- Создайте БД `excametr_database`
- Импортируйте файл `database.sql`

### 3. Настройте подключение в коде
- Откройте `DB.cs`
- Укажите строку подключения: `Server=localhost;Database=excametr_database;Uid=root;Pwd=;`

### 4. Запустите приложение
- Откройте `ExcaMetr_Ver_0.0.sln` в Visual Studio
- Нажмите `F5`
